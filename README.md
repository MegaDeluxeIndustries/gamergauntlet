## Features

- Muting audio
- Switching to a scene
- Hiding a source
- Toggle your stream on and off
- Toggle your recording on and off
- Pause and resume your recording
- Cross-platform
  - App:
    - Android
    - (iOS) **coming soon**
  - Computer:
    - Linux
    - Windows
    - macOS
